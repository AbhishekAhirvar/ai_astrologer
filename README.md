# AI Astrologer

An AI-powered Vedic Astrologer chatbot built with Python and Gemini.

## Features
- Vedic Chart Generation (D1, D9, D10, D12)
- Nakshatra Analysis
- AI-based predictions and explanations
- Interactive UI using Gradio

## Setup
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set your Gemini API key in a `.env` file:
   ```
   GOOGLE_API_KEY=your_api_key_here
   ```
4. Run the application:
   ```bash
   python app.py
   ```
